Maternal Health Risk Stratification

ML + XAI Using Accessible Vitals

← Back to Projects

Main Technology Stack

Python, Scikit-learn, Pandas, SHAP, UCI Maternal Health Risk dataset.

Brief Description

A machine learning system that stratifies maternal health risk (low / mid / high) using easily accessible vitals such as blood pressure, blood sugar, and heart rate, combined with SHAP-based explainability to make the model's reasoning transparent. Given the limitations of the underlying dataset, the project is intentionally framed as a "risk stratification using accessible vitals" tool rather than a clinical diagnostic system, making it suitable as a supportive screening aid in low-resource settings.

Challenges Faced

  • Working with a dataset limited in size and clinical depth, which required careful framing to avoid overstating diagnostic claims.
  • Balancing model performance with interpretability so the risk categories remain clinically meaningful.
  • Structuring a literature review and clinical grounding phase before any modeling began.

Future Improvements

  • Incorporate additional real-world clinical datasets to strengthen generalizability.
  • Add a simple web-based interface for healthcare workers to input vitals and receive a risk estimate.
  • Explore counterfactual explanations (e.g. DiCE) to suggest actionable vitals changes.
