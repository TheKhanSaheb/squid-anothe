Jute Leaf Disease Classification

Multi-Backbone Deep Learning with Triple XAI

← Back to Projects

Main Technology Stack

Python, TensorFlow/Keras, ResNet50, EfficientNetB4, ConvNeXt-Tiny, InceptionV3, MobileNetV2, Optuna, Grad-CAM, SHAP, LIME, Kaggle T4 GPU.

Brief Description

A full modular pipeline for classifying jute leaf diseases across 5 classes using the Harvard jute leaf dataset (1,390 images). The project compares five CNN backbones, applies Optuna-based hyperparameter optimization (30 trials) to find the best-performing configuration, and uses a triple explainability approach — Grad-CAM, SHAP, and LIME — to make model predictions interpretable for agricultural use cases. The work is aimed at publication in journals such as MDPI Plants/Agriculture or Frontiers in Plant Science.

Challenges Faced

  • Working with a relatively small dataset (1,390 images) while avoiding overfitting across 5 different backbones.
  • Running a 30-trial Optuna search within limited Kaggle GPU quota.
  • Reconciling three different XAI methods (Grad-CAM, SHAP, LIME) into a consistent, comparable interpretation of model behavior.

Future Improvements

  • Expand the dataset with more field-collected images to improve real-world robustness.
  • Package the best-performing model into a lightweight mobile app for farmers.
  • Add multi-crop disease support beyond jute.
